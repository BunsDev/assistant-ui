The `AssistantRuntime` is the root runtime of the application. It has been updated to be generic, which means it can be used with a specific type that extends `AssistantRuntime`.

### `useAssistantRuntime`

```tsx
import { useAssistantRuntime } from "@assistant-ui/react";

const runtime = useAssistantRuntime<YourSpecificRuntimeType>();
```

### Tool UI Registry

The tool UI registry is part of the assistant runtime. It allows you to display custom UI for tool calls, enabling generative UI.

#### `useToolUIs`

```tsx
import { useToolUIs } from "@assistant-ui/react";

const toolUIs = useToolUIs();
const webSearchToolUI = useToolUIs((m) => m.getToolUI("web_search"));
```

Please replace `YourSpecificRuntimeType` with the specific type that extends `AssistantRuntime` in your application. If you are using the default `AssistantRuntime`, you can use `AssistantRuntime` as the type.
