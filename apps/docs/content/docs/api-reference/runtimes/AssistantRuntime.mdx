The `AssistantRuntime` is the root runtime of the application. It is now used as a generic parameter in several places.

### `useAssistantRuntime`

```tsx
import { useAssistantRuntime } from "@assistant-ui/react";

const runtime = useAssistantRuntime();
```

<ParametersTable {...AssistantRuntime} />

The `useAssistantRuntime` function now accepts a generic parameter `T` that extends `AssistantRuntime`. This allows you to specify the type of runtime you are using.

```tsx
import { useAssistantRuntime } from "@assistant-ui/react";

const runtime = useAssistantRuntime<MyCustomRuntime>();
```

### Tool UI Registry

The tool UI registry is part of the assistant runtime. It allows you to display custom UI for tool calls, enabling generative UI.

#### `useToolUIs`

```tsx
import { useToolUIs } from "@assistant-ui/react";

const toolUIs = useToolUIs();
const webSearchToolUI = useToolUIs((m) => m.getToolUI("web_search"));
```

The `AssistantRuntimeProvider` now also accepts a generic parameter `T` that extends `AssistantRuntime`. This allows you to provide a specific type of runtime to the rest of your app.

```tsx
import { AssistantRuntimeProvider } from "@assistant-ui/react";

<AssistantRuntimeProvider<MyCustomRuntime> runtime={myCustomRuntime}>
  {/* Your app */}
</AssistantRuntimeProvider>
```

The `useRemoteThreadListRuntime` function now accepts a generic parameter `T` that extends `AssistantRuntime`. This allows you to specify the type of runtime you are using.

```tsx
import { useRemoteThreadListRuntime } from "@assistant-ui/react";

const runtime = useRemoteThreadListRuntime<MyCustomRuntime>(options);
```

The `RemoteThreadListOptions` type now accepts a generic parameter `T` that extends `AssistantRuntime`. This allows you to specify the type of runtime you are using.

```tsx
import { RemoteThreadListOptions } from "@assistant-ui/react";

const options: RemoteThreadListOptions<MyCustomRuntime> = {
  runtimeHook: myCustomRuntimeHook,
  adapter: myAdapter,
};
```

The `RemoteThreadListHookInstanceManager` class now accepts a generic parameter `T` that extends `AssistantRuntime`. This allows you to manage instances of a specific type of runtime.

```tsx
import { RemoteThreadListHookInstanceManager } from "@assistant-ui/react";

const manager = new RemoteThreadListHookInstanceManager<MyCustomRuntime>(myCustomRuntimeHook);
```
